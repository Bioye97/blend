Modules
=======

BLEND modules are called from the main ``blend`` program:

.. code-block:: sh

   blend <module-name> [module-options]

.. toctree::
   :maxdepth: 1

   blend

Current module names
--------------------

* ``docs``: show documentation for a module
* ``monotone``: check or enforce xy-monotone polygon boundaries
* ``window1d``: generate 1-D window weights
* ``window2d``: generate 2-D window weights
* ``window3d``: generate 3-D window weights
