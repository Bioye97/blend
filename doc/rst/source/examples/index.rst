Examples
========

Examples are stored in ``doc/examples`` using the GMT-style numbering pattern
``ex01``, ``ex02``, and so on.

Current examples
----------------

* ``ex01``: 1-D cosine taper.
* ``ex02``: 1-D trapezoid taper.
* ``ex03`` through ``ex10``: 2-D polygon tapers over regular polygons.
* ``ex11``: 2-D isotoxal-star support.
* ``ex12``: 3-D isotoxal-star support.
* ``ex13``: South America polygon monotone conversion.
